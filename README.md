# Ґаудія Ґітанджалі

Пісенник [Kirtan Guide](https://github.com/scsm-ua/kirtan-guide-en) з автоматичною транслітерацією віршів українською, без перекладу змісту.

## Building

To build markdown songs to JSON format:

```bash
npm run build
```

This will process all `.md` files from the `songs/` directory and generate `.json` files in the `build/songs/` directory.
